Common Issues

Nearby stations not showing

  • Enable location permissions for FuelGO.
  • Turn on GPS/Location Services.
  • Check internet connectivity.

Prices seem outdated

  • Pull to refresh the app/map.
  • Check selected fuel type.
  • Station updates can lag in fast-moving periods.

Subscription Support

  • Subscriptions are billed via Apple App Store.
  • Monthly and annual plans auto-renew by default.
  • Manage/cancel in iOS Settings → Apple ID → Subscriptions.
  • Use in-app Restore Purchases if premium access is missing.

Restore Purchases (Step-by-step)

  1. Open FuelGO.
  2. Go to Settings (or Profile).
  3. Tap Restore Purchases.
  4. Wait for entitlement confirmation.
  5. If needed, verify you are signed in to the original Apple ID used to buy.
